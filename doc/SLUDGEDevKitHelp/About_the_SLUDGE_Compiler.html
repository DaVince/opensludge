About the SLUDGE Compiler

Running the SLUDGE compiler program yourself, simply by double-clicking on the SLUDGE compiler icon, will have no effect. This is because the program requires a SLUDGE project file on which to operate. You can use the SLUDGE compiler to compile an existing SLUDGE project in four ways...

1) Right-click a SLUDGE project (with the extension .SLP) and choose "Compile" from the menu which appears.

2) Run the SLUDGE project manager and open a SLUDGE project. Then hit either the Compile or Compile and run button.

3) Double-click a SLUDGE project to load it into the SLUDGE project manager and then hit either the Compile or Compile and run button.

4) Drag a SLUDGE project onto the SLUDGE compiler icon.

The SLUDGE compiler is completely automated and will tell you of any errors which it finds with your project. If the compiler closes without warning you of any errors then your project has been compiled successfully.
