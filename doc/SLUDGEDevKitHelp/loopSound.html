loopSound

Syntax:

loopSound (thisSoundFile);

Purpose:

Plays a sound repeatedly, starting immediately. If the parameter isn't a file handle the code has no effect. Supported sound formats are Windows wave files (.WAV), MPEG layer 3 (.MP3) and Ogg Vorbis (.OGG).

Return value:

No return value.
