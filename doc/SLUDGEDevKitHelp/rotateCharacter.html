rotateCharacter

Syntax:

rotateCharacter (thisCharacter, degrees);

Purpose:

Rotates thisCharacter by degrees. A value of 0 degrees means that the character will be drawn normally, while 180 places it upside down.

Return value:

TRUE if thisCharacter is an object type which is currently represented on the screen as a character, otherwise FALSE (i.e. TRUE if the call to the function was a success).
