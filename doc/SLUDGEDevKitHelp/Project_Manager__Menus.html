Project Manager: Menus

In the following the actions accessible over the SLUDGE Project Manager menus are described.

File menu:

New:

Opens a file save box, where you can choose a location and name for the project file you wish to create. The new file will be created containing no script files and its project settings will be set to default values. Then the project settings dialog is launched for you to make your first changes.

Open:

Opens a file open box which lets you locate the project file which you want to load.

Save:

Saves the current project. The file from which the project was loaded (or specified when the project was created) will be overwritten. To save the project into a new location, use Save as... instead.

Save as:

Lets you save the existing project to a different filename. Note that if you save the project file to a directory other than the directory in which the old file was located, the file locations in the Project Files tab will be wrong.

Exit:

Quits the SLUDGE project manager.

Edit menu:

Preferences:

Opens the Preferences dialog box, where you can manipulate global preferences for the Project Manager.

Project menu:

Project properties:

Opens the Project Settings dialog box, where project specific settings are made.

Compile:

Runs the SLUDGE compiler and, errors permitting, creates a compiled SLUDGE game (.SLG) file. The name of the file will be that which is specified in the Project Settings dialog. Any warnings or errors will be shown in the Compiler Errors tab in the main window of the project manager.

Run:

Runs the game. Only works if the project has been compiled since it was loaded.

Help menu:

About:

Displays a pop-up dialog box giving the version number and other information about the Project Manager.

SLUDGE documentation:

Opens this help file.
