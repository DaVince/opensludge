Project Manager: "Project management" Section

This area contains eight buttons which allow you to work with entire projects.

Create new project:

Opens a standard Windows file save box, where you can choose a location and name for the project file you wish to create. The new file will be created containing no script files and its project settings and project properties will be set to default values.

Open existing project:

Opens a standard Windows file open box which lets you locate the project file which you want to load.

Save project:

Saves the current project. The file from which the project was loaded (or specified when the project was created) will be overwritten. To save the project into a new location, use the Save as... option on the File menu instead.

If a project is loaded and this button is unavailable, the file is up-to-date.

Project properties:

Opens the project properties dialog box which lets you edit the project's width, height, quit message and other video-related settings.

Compile project:

Runs the SLUDGE compiler and, errors permitting, creates a compiled SLUDGE game (.SLG) file. The name of the file will be that which is specified in the Project settings section.

Compile and run project:

Identical to the Compile project button, except that the compiled game - should one be successfully created - will be run immediately.

About:

Displays a pop-up dialog box giving the version number of the project manager and a link to the Hungry Software SLUDGE website.

Help!:

Opens this help file.
