playSound

Syntax:

playSound (thisSoundFile);

Purpose:

Plays a sound once, immediately. If the parameter isn't a file handle the code has no effect. Supported sound formats are Windows wave files (.WAV) and Ogg Vorbis (.OGG).

Return value:

No return value.
