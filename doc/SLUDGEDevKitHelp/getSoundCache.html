getActiveSounds

Syntax:

getActiveSounds ();

getSoundCache (); * This function used to be called getSoundCache.

Purpose:

Finds out which samples are currently playing.

Return value:

The function returns a stack containing one file handle for each sample which is currently playing. You can subsequently loop through this stack to change the volume of or stop all active samples.
