registryGetString

Syntax:

registryGetString (thisRegistryKey, thisElement);

Purpose:

Gets the specified string from the HKEY_LOCAL_MACHINE section of the Windows registry. SLUDGE can't currently write to the registry, but this function is very useful for retrieving settings written there by, for example, an installer or another program.

Return value:

If the HKEY_LOCAL_MACHINE section of your Windows registry contains the key thisRegistryKey and that key contains the element thisElement, that string will be the return value. If not, the return value will be NULL.

Example:

var myVersion = registryGetString ("Someone Software\\Potato Quest", "version");
