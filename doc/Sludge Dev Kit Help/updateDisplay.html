updateDisplay

Syntax:

updateDisplay (onOff);

Purpose:

Turns on or off whether the background image and sprites are drawn to the window every frame or not. Turning off updateDisplay can speed up playback of movies, for example, but remember to turn it back on afterwards. To turn display updating on, onOff should be TRUE - to turn it off, onOff should be FALSE.

Return value:

No return value.

This function is unavailable to unregistered users of the development kit. To use this function, and for a heap of other benefits, please register your copy of the SLUDGE development kit.
