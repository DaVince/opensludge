setCharacterAntiAliasing

Syntax:

setCharacterAntiAliasing (thisCharacter, onOrOff, softnessX, softnessY);

Purpose:

Changes the anti-aliasing settings for thisCharacter, provided it has already been added to the scene using addCharacter. onOrOff defines whether anti-aliasing is enabled or disabled. softnessX and softnessY define the blurriness of the character when it is drawn each frame (these are ignored if onOrOff equates to FALSE). Take a look at some images of anti-aliasing in action to see the effects of the different softness values.

Please note that the character's default anti-aliasing settings are not modified; if this character is added again later, the default settings will be used rather than these modified ones.

Return value:

Returns TRUE if the anti-aliasing settings were altered (ie. if the character was in the scene), FALSE if not.
