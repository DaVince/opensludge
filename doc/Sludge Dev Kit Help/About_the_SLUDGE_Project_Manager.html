About the SLUDGE Project Manager

You can start the SLUDGE project manager in the following ways...

1) Choose "SLUDGE Project Manager" from the "SLUDGE by Hungry Software" section of your Windows Start menu.

2) Find the folder into which you installed the SLUDGE Development Kit and double-click the "SLUDGE Project Manager.exe" icon.

3) Double-click a SLUDGE project file, with the extension ".SLP".

The SLUDGE project manager window will appear, looking like this:



The three sections which make up the window are labelled Files in this project (which lists the contents of the current project), Project management (which contains buttons for handling projects) and Project settings (which shows the project file currently loaded, its name and the output filename). There are also four menus at the top of the window. Some functionality can be accessed from either the menus or buttons on the window.
