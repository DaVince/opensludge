Project Manager: "Project settings" Section

This area contains three text areas.

Input filename:

The location and name of the file which is currently open. You cannot edit this text. Instead, to load a new file, press the "Open existing project" button in the Project management area of the window. To save the project to a new filename, go to the "File" menu and choose the "Save as..." option.

Project name:

The text which will be visible in the title bar of the game (and also the ALT-TAB window and taskbar) when it's running.

Output file:

The name of the project file to be created by the SLUDGE compiler when the project is compiled. The extension ".SLG" will be added automatically - please don't add this yourself. The name entered here is also used for a file containing all the strings in your game - see the Spellchecking the Strings in your Game topic.
